# Development

> In order to use this app you will need perl 5 and the following perl modules
installed:

* Plack
* LWP::UserAgent
* JSON
* Text::Template
* LWP::Protocol::https

> Run tests:

```
prove -Ilib t
```

# How to use

```
plackup [--port PORT_NUMBER] -Ilib
```
